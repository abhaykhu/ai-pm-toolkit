# Dev Task — Interview Guide & Template

## Interview Questions

### Task Description
- What needs to be done? (technical description)
- Why is this needed? (context/motivation)
- Is this part of a larger feature or epic?

### Acceptance Criteria
- How do we know this is done?
- Any specific requirements for the implementation?
- Are there edge cases to handle?

### Technical Context
- Any relevant code pointers or architecture notes?
- Dependencies on other tasks or systems?
- Estimation guidance — is this small, medium, or large?
